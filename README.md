# The Amazing Spider-Man Web Experience

A cinematic web experience showcasing The Amazing Spider-Man through an interactive sticky scroll interface.

## Description

This project creates an immersive web page featuring Spider-Man's world with smooth scene transitions, parallax effects, and responsive design. Users can scroll through four key scenes: Origin, Agility, The Leap, and Legend, each with cinematic visuals and animations.

## Features

- **Sticky Scroll Experience**: 500vh scroll with scene-based transitions
- **Cross-Fade Animations**: Smooth opacity transitions between scenes
- **Responsive Design**: Optimized for desktop and mobile
- **Custom Cursor**: Interactive cursor effects
- **Loading Animation**: Web-shooter initialization loader
- **Scroll-Reveal Elements**: Quotes, movie stills, and feature cards with reveal animations

## Technologies Used

- HTML5
- CSS3 (Animations, Flexbox, Grid)
- JavaScript (ES6+)


## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/yourrepo.git
   cd yourrepo
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Usage

### Local Development

1. Start a local server (e.g., using Python, Node.js, or VS Code Live Server):
   ```bash
   # Using Python 3
   python -m http.server 3000

   # Or using Node.js (if you have a server setup)
   # npm start (if configured)
   ```

2. Open `http://localhost:3000/tasm-v2.html` in your browser.

3. Let the loader complete, then scroll through the scenes.

### Testing

Run the verification script (if available) or manually test as per `verify.md`.
